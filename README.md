# Multilingual_chatbot_guvi
Multilingual chatbot is Created for GUVI . It is Pre-Trained GPT model to answer user queries.The user can ask queries about GUVI and chatbot will answer it. if it is non-english language, the chatbot give the user to choose language and the chatbot should translate the input to english using RAG model, get a response from the GPT model and translate back to the original language. The text file contain all details about the GUVI is first loaded and preprocessing step is done which includes Tokenization. Necessary package like deep translator, google translator, sentence transformer etc.Distil-GPT2 model is choosen to perform this multilingual chat. then the user questions are gathered in the bot and the bot will response with its answer in whatever language it is asked.
